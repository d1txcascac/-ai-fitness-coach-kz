AI Fitness Coach KZ 🇰🇿Real-time Exercise Form Correction using Human Pose Estimation.Project OverviewAI Fitness Coach KZ is an open-source initiative aimed at making high-quality fitness coaching accessible through Computer Vision. The project utilizes the YOLOv11-pose model to analyze human body keypoints in real-time, providing immediate feedback on exercise technique to prevent injuries and improve performance.Core TechnologiesComputer Vision: Ultralytics YOLOv11-pose for detecting 17 body keypoints.Real-time Analysis: OpenCV for frame processing and biomechanical angle calculation.LLM Integration: GPT-4o-mini for generating personalized corrective feedback based on pose data.Framework: Streamlit for web-based prototyping and demonstration.Technical FeaturesJoint Angle Detection: Automated calculation of knee and hip angles (e.g., ensuring >90° depth in squats).Injury Prevention: Heuristic-based monitoring to detect unstable posture or incorrect alignment.Cross-Platform: Designed for deployment via Telegram Web Apps to ensure accessibility on mobile devices.Installation & UsageClone the repository:Bashgit clone https://github.com/[d1txcascac]/ai-fitness-coach-kz.git
Install dependencies:Bashpip install -r requirements.txt
Run the demo:Bashstreamlit run streamlit_demo.py
